


## Array.max()

[CDN link](https://unpkg.com/@khalyomede/prototype-storm@0.1.0/dist/array/max.js)

Returns the maximum among all the numbers of the array.

**return**
Number

```javascript
const array = [1, 2, 3];
array.max(); // 3
```
